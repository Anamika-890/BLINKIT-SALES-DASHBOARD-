# Blinkit Sales Dashboard (Excel)

## Project Overview

This project is an interactive sales dashboard created in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers. The dashboard analyzes Blinkit sales data and provides insights into product ratings, outlet performance, and sales distribution.

## Dashboard Preview

The dashboard includes:
* Total Sales KPI
* Average Rating KPI
* Product Count KPI
* Rating by Item Type Chart
* Sales by Outlet Type Chart
* Sales by Outlet Size Chart
* Interactive Outlet Type Slicer
